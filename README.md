# 桌面宠物 - 大龙虾 🦞

一个可爱的透明窗口桌面宠物，基于 Electron + Vue3 开发。

## 功能特点

- ✨ 透明无边框窗口
- 🖱️ 可拖拽移动
- 🦞 可爱的大龙虾形象
- 💬 点击互动，随机对话
- ❤️ 心情系统
- ⚡ 精力系统（随时间减少）
- 📌 始终置顶显示
- 🚫 不占用任务栏

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行

```bash
npm start
```

## 操作说明

- **拖拽**: 在宠物身上任意位置拖动
- **点击**: 点击大龙虾，它会随机说话
- **关闭**: 鼠标悬停在宠物上，右上角会出现关闭按钮

## 技术栈

- Electron - 桌面应用框架
- Vue 3 - 前端框架
